Mac下部署hexo

搞了几天终于把hexo在mac上部署完成了，期间遇到了好多小问题，趟了一遍记下一点，备忘

要部署hexo需要在电脑上先安装好git和node.js，而MAC电脑需要使用Homebrew命令行工具来进行安装。

可是Homebrew也需要安装才可以使用，安装方式如下

在mac中运行终端，输入以下命令

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

由于网络的不稳定安装时可能会遇到443的报错，在无法翻墙的情况下，可查询国内安装homebrew的方法

安装好homebrew之后可直接在终端中利用homebrew安装node.js

brew install node

安装好之后可以使用 node -v来检查是否安装成功

然后再安装git

brew install git

准备工作做好之后开始安装hexo

首先建立一个blog文件夹，用来存放所有hexo的博客系统及文件

我用Finder在文稿目录下新建了一个hexo文件夹

进入文件夹

$ cd ~/Docments/hexo

用npm安装hexo命令行工具

sudo npm install -g Hexo-cli

安装完成后，在该目录下执行博客初始化操作

hexo init

命令执行完毕后，会在该目录下生成blog文件夹，进入该blog文件夹下，可以看到生成的blog系统文件

在该blog目录下执行运行命令

hexo s

当看到如下输出

则表示本地运行成功，此时可在浏览器输入http://localhost:4000进行hexoblog预览

完成以上本地部署后，开始进行本地推送至github的配置,

先注册用户账户

然后在github中新建Repository

图中带*的需要保持一致，即Repository name 前缀和owner 一致为se7eneo.github.io

  • 配置客户端git添加ssh key 到github

进入用户目录下的.ssh目录下(.ssh目录为隐藏目录，如果想在finder中看到需要按command+shift+.)

cd ~/.ssh

执行以下命令

  1. 认证Git账号

    配置账号：

    git config --global user.neme “your name”

    git config --global user.email “yourname@mail.com”

    生成ssh登陆密钥

    ssh-keygen -t rsa -C yourname@mail.com

    如果已经创建过会出现 Overwrite（y/n）？，如果没有创建过会要求我们输入密码，然后一路回车，执行完成后会在～/.ssh/id_rsa.pub目录下生成需要使用的key

    在终端中利用命令行显示key并复制（或者在finder中用软件打开后复制key）

    cat ~/.ssh/id_rsa.pub

  2. 获取SSH接口

    登陆github，找到setting，先点击SSH and GPG keys 然后再点击New SSH key进入到配置SSH key的页面

    然后将复制好的key粘贴到 key对话框内

    点击 Add SSH Key即可

    添加成功后，可以在命令提示符输入

    ssh git@github.com

    查看是否连接正常

  3. 安装hexo部署git,命令行进入到本地blog目录下

    cd ~/Documents/hexo/blog

    执行部署git命令

    npm install Hexo-deployer-git --save

  4. 修改hexo配置文件_config.yml

    vim ~/Documents/hexo/blog/_config.yml

    修改deploy部分为

    修改完成wq保存

    然后就可以把本地博客推送到github了

    hexo g

    hexo d

    打开se7eno.github.io就可以看到博客了

  5. 写博客

    在命令行中执行 hexo new blogname.md

    会在～/Documents/hexo/blog/source/_post目录下生成blogname.md文件，打开这个文件即可写博客，然后本地保存后重新执行

    hexo g

    hexo d

    就可以把blog推送到github

    • 申请域名指向博客，就可以直接使用域名访问
    • hexo支持很多模版样式，可以去官网选择自己喜欢的
